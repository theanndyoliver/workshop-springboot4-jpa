# Estudos de Spring Boot

Repositório com projeto desenvolvido durante o curso "Java COMPLETO Programação Orientada a Objetos + Projetos".

## Conteúdo

* Criação de APIs REST
* Estruturar camadas lógicas: resource, service, repository
* Configurar banco de dados de teste (H2) 
* Integração com banco de dados
* CRUD - Create, Retrieve, Update, Delete
* Modelo Relacional - Um para Um, Um para Muitos,Muitos para Um e Muitos para Muitos
* Tratamento de exceções

## Tecnologias

* Java 25
* Spring Boot 4
* JPA/Hibernate
* PostgreSQL
